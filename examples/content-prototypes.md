# PART B — FEATURE (Magazine) · "חוצפה / Chutzpah expressions" · slang

Real Magazine card. 4 idioms; **1 dropped by the safety gate** (see below). Hero atom = **carousel** (not video).

## Idiom-reveal video (11 screens · ~33s) — the new signature format
Per idiom: **literal screen (bait, greyed)** → **reveal screen (real meaning, bold)** → **example**. Uses the 3 safe idioms.
- Cover: חֻצְפָּה יִשְׂרְאֵלִית / *Israeli chutzpah — 3 expressions*
- דָּחַף אֶת הָאַף → "pushed his nose in"? → **= butted in / stuck his nose in uninvited** → ex: דָּחֲפָה אֶת הָאַף לַשִּׂיחָה שֶׁלָּנוּ
- לֹא דּוֹפֵק חֶשְׁבּוֹן → "doesn't knock an account"? → **= does as he pleases, doesn't care** → ex: פָּנָה יְשִׁירוֹת לַמְּנַהֵל
- עָשָׂה שְׁכוּנָה → "made a neighborhood"? → **= [SEE CORRECTION: did a sloppy/low-quality job]** → ex: עָשָׂה שְׁכוּנָה מוּל כָּל הַמִּשְׂרָד
- Follow beat: עֲקֹב לְעוֹד בִּיטּוּיִים / *Follow for more Hebrew slang*

## Carousel (6 slides · the HERO)
1. **Cover** (ON photo): חֻצְפָּה בְּעִבְרִית / *Speaking with chutzpah — 3 expressions*

3. **Idiom** (panel): לֹא דּוֹפֵק חֶשְׁבּוֹן — lit. "doesn't knock an account" → does as he pleases
4. **Idiom** (panel): עָשָׂה שְׁכוּנָה — lit. "made a neighborhood" → [corrected: did a sloppy job]
5. **Recap** (panel): שָׁלוֹשׁ דְּרָכִים לוֹמַר "חֻצְפָּה" — 3 idioms + translit stacked
